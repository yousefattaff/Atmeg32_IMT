
**Atmega32_IMT**

This repository contains drivers and example applications developed for the **ATmega32 microcontroller**. The primary goal of the project is to build, test, and demonstrate reusable embedded drivers while improving modularity and hardware abstraction.

The project includes multiple low-level drivers written in **C** to interface with common microcontroller peripherals. Each driver is accompanied by small test applications that validate its functionality and demonstrate practical usage.

The repository is still under active development. Additional drivers and improvements are planned, including refactoring existing drivers to follow a **post-build / linker-time configuration approach**. This change aims to make the drivers more modular and adaptable, allowing them to be reused across different hardware components and configurations without requiring source-level modifications.

Overall, the project focuses on strengthening embedded systems fundamentals such as **peripheral interfacing, driver architecture, modular design, and hardware abstraction**.
